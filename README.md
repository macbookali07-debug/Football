## 🔧 Кастомизация

### Изменение цен
Отредактируйте объект `fieldPrices` в `script.js`:
```javascript
const fieldPrices = {
    '5x5': { base: 1500, peak: 2000, evening: 2500 },
    '8x8': { base: 2500, peak: 3000, evening: 3500 },
    '11x11': { base: 4000, peak: 4500, evening: 5000 }
};
```

### Изменение занятых слотов
Отредактируйте объект `bookedSlots` в `script.js`:
```javascript
const bookedSlots = {
    '2024-01-15': ['10:00', '11:00', '18:00', '19:00'],
    // добавьте даты и занятые времена
};
```

### Цветовая схема
Измените CSS переменные в `styles.css`:
```css
:root {
    --primary-color: #2ecc71;
    --secondary-color: #27ae60;
    --accent-color: #e74c3c;
    /* ... */
}
```
