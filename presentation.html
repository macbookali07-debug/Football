Шейит-Дөбө Arena

Аренда футбольных полей

Профессиональный веб-сайт для онлайн-бронирования футбольных полей с современным дизайном и полным функционалом

О проекте

  • Сайт предназначен для аренды футбольных полей в городе Ош
  • Позволяет клиентам бронировать поля онлайн на удобное время
  • Имеет современный дизайн с красивыми анимациями
  • Работает на чистом JavaScript без внешних фреймворков
  • Полностью адаптирован для мобильных устройств
  • Интегрирован с Google Maps для показа местоположения

Основные возможности

Автоматический слайдер

Fade анимации, переходы между фотографиями полей с управлением точками-индикаторами

Календарь бронирования

Выбор даты с ограничением на прошедшие даты и динамическое ценообразование

Онлайн-оплата

Модальное окно оплаты с валидацией карты и авто-форматированием номера

Интерактивная карта

Google Maps интеграция с кастомным маркером и информационным окном

Дизайн и UX

  • Современный дизайн с градиентами, анимациями и эффектами
  • Адаптивная верстка (Responsive Design) для всех устройств
  • Плавные анимации при загрузке и взаимодействии
  • Мобильное меню с бургер-анимацией и переходами
  • Параллакс эффекты при прокрутке страницы
  • Экран загрузки с красивой анимацией футбольного поля
  • Анимированные частицы на фоне для динамизма

Технологии и инструменты

HTML5 CSS3 JavaScript Google Maps API Font Awesome Google Fonts
  • HTML5 — семантическая разметка
  • CSS3 — современные стили (Grid, Flexbox, анимации)
  • Vanilla JavaScript — интерактивность без фреймворков
  • Нет зависимостей — чистый, быстрый код

Система бронирования

  • Выбор типа поля: 5x5 или 8x8
  • Динамическое ценообразование в зависимости от размера поля
  • Отображение занятых временных интервалов
  • Валидация выбора доступного времени для бронирования
  • Процесс оплаты с имитацией обработки платежа
  • Работь с 08:00 до 23:00 по умолчанию
